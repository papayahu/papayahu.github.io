官方plugin總覽

ReMoods Theme

  • Night Mode模式推薦：
  • red0 紅
  • Yellow+ 淺綠
  • Yellow-Green- 綠
  • Cyan-Blue- 淺藍
  • Cyan-Blue210 藍（正在用）
  • 添加目錄：在每個記事的Markdown書寫欄開頭加入 [[toc]]

Emoji Picker

  • 使用 :: 開啟emoji

Favorites

  • 右鍵：Add note to favorites
  • 工具 - Favorites

Freehand Drawing

  • 上方隱藏功能欄 🖊

Joplin Calender

  • 記事 / 待辦事項的標題欄最右側按鈕：開啟 / 關閉日曆

MarkMap

  • 記事 / 待辦事項的標題欄右二按鈕：開啟mindmap

Random Note

  • 上方隱藏功能欄右六按鈕：開啟隨機一則Note

RevealJS Integation

  • 上方隱藏功能欄右四按鈕：開啟幻燈片

Rich Markdown

  • 工具 - 選項 - 延伸模組 - Rich Marchdown 開啟所需功能

Rubi and Furigana

  • 假名前後使用rt標籤，漢字前後使用ruby標籤。範例：今日きょう

Slash Commands

  • 使用 / 插入日期、時間、日曆

Suitcase

  • 將文字反白
  • 編輯 - Capitalization - 選擇所需格式

Text Colorize

  • 工具 - Text Colorize：更換內文顏色或添加高光
  • 清除已儲存的顏色：右鍵點選之
  • #f37f7f 紅色標記色碼

LH: [Welcome to Joplin!](#welcome-to-joplin)
GI: [welcome-to-joplin](:/524d6874790f4e33a1dd2f5b2463c6cb#welcome-to-joplin)
